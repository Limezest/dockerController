# dockerController
Web interface for Docker

![Dashboard view](http://i.imgur.com/ABEBt6O.png)


### How to make it work

//  Install the modules

$ npm install



//  Create the log files

$ touch logs/{node,gulp}/{err,out}.log



//  Actualize the frontend

$ cd src/public

$ gulp



//  Start the backend

$ cd ../..

$ npm start
